---
description: Setup the development environment for DavomatAI
---

1. Create a Python virtual environment in the `backend` directory.
2. Install backend dependencies: `pip install -r backend/requirements.txt`.
3. Run the model download script: `python backend/download_models.py`.
4. Initialize the database: `python backend/init_db.py`.
5. Install frontend dependencies: `cd frontend && npm install`.
